## MUTE\_CHANGED

MUTE state (On/Off) has changed.


### Name

`MUTE_CHANGED ()`


| Parameter | Type | Description                                                           |
| --------- | ---- | --------------------------------------------------------------------- |
|           | BOOL | True/False                                                            |
| Output    | INT  | Indicates a change in mute state for the specified output binding ID. |


### Returns

`None`
